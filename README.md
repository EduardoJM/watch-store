## How to Run Locally

### 1. Clone the Repository

```sh
https://github.com/EduardoJM/watch-store.git
cd watch-store
```

### 2. Install the dependencies

```sh
yarn
```

### 3. Run the development version

```sh
yarn dev
```

## How to run tests

### With Jest

```sh
yarn test
```

#### In watch mode

```sh
yarn test:watch
```

#### With code coverage

```sh
yarn test:coverage
```
